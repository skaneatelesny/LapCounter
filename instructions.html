Instructions - Set Up

Laps

Enter the number of laps to be run. Example: 3200m outdoor = 8 laps.

Hips

Enter the number of runners in the race. Actually, enter the highest 'hip number' in the race. Ignore any missing runners ('scratches'). They just never finish. The app builds a 'lap table' as you enter laps and hips. The lap table tells you which runner has how many laps to go. After you enter both laps and hips the app jumps to the run screen.

Instructions - Run Screen

There are two jobs once you get to the run screen.

  1. Start the clock
  2. Enter hip numbers as the runners go by.

Starting the clock

You can either hit 'START' or enter '0'.
When all runners have finished the clock stops and you will have a completed finishing order list with times. It is important to hit 'enter' at the moment the runner crosses the finish line if you want accurate times.

Entering Data

There are three ways to enter data.<>br

  1. Screen keypad
  2. External keypad
  3. Voice recognition

Screen keypad

If no external keypad is connected, a screen keypad pops up automatically. This can be used, but consider having two people for most races. Have one person call out numbers, second person punch them in. If race traffic is hot and heavy, have the first person scribble them down on paper while calling them out. That way the keyboard person can catch up when needed. The keyboard works, but realize that your eyes can end up moving rapidly between the race and the keypad.

Laps

  1. Enter the number of laps to be run.
  2. Example: 3200m outdoor = 8 laps.
  3. Press Enter to continue.

Starting a Race

  1. Enter number of laps.
  2. Enter number of athletes.
